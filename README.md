# Next.js Resume

## Description

This project is a personal resume website built using Next.js and Tailwind CSS.

## How to Run

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open your browser and go to:

```
http://localhost:3000
```

## Features

- Resume built using Next.js App Router
- Resume written in JSX inside page.js
- Styled using Tailwind CSS
- Includes Education
- Includes Professional Experience
- Includes Leadership & Extracurricular Activities
- Includes Skills, Certifications, and Interests
- Interactive show/hide buttons for each major resume section using React state.
